# Репутация

## Назначение

Репутация — накопительный показатель доверия для подрядчиков и экспертов. Для компаний-заказчиков показатель не рассчитывается.

## Сущности

- `reputation_rules` — изменяемая стоимость событий;
- `reputation_events` — неизменяемый журнал начислений;
- `reputation_summary` — агрегат для быстрого интерфейса;
- отзывы и ответы опросника — отдельные данные обратной связи.

## Бизнес-логика

- Начисление всегда создаёт событие с источником.
- Повторное событие от одного источника не должно дублировать баллы.
- Значения баллов меняются в таблице правил, а не в коде.
- Пересчёт выполняется отдельной осознанной операцией.
- Публично показываются баллы, число отзывов и рекомендаций без звёзд.
