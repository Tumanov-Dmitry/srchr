# Избранное и коллекции

## Назначение

Избранное — персональный слой пользователя поверх компаний, экспертов, кейсов и статей. Оно не изменяет исходные объекты.

## Бизнес-логика

- Один объект нельзя добавить одному пользователю дважды.
- Добавление и удаление выполняются одной кнопкой-состоянием.
- Можно закрепить не более четырёх карточек суммарно.
- Закреплённые отображаются первыми, затем новые по дате добавления.
- Избранное можно распределять по пользовательским коллекциям.

## Недоступные объекты

При добавлении сохраняется snapshot: название, описание, изображение и тип. Если исходный объект удалён или скрыт, карточка остаётся в избранном как недоступная, но её можно удалить.
