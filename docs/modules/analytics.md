# Аналитика

## Назначение

Модуль фиксирует продуктовые события и показывает сводные показатели владельцам сущностей и администраторам.

## Бизнес-логика

- Событие содержит тип действия, пользователя, целевой объект и технический контекст.
- Публичный просмотр не должен давать пользователю возможность подменить владельца объекта.
- Владелец видит статистику только доступных ему организаций и сущностей.
- Администратор видит агрегаты платформы.
- Технические и чувствительные поля не отображаются в публичном интерфейсе.

## Интерфейс

- `/dashboard/analytics`
- `/dashboard/expert/analytics`
- `/dashboard/organization/analytics`
- детальная аналитика материалов, задач и мероприятий
- `/admin/analytics`

## Связи

Аналитика принимает события от публичных каталогов, материалов, задач, мероприятий, избранного и запросов стоимости.
