# Подрядчики

## Назначение

Подрядчик — это организация с профилем исполнителя. Модуль включает редактирование профиля, услуги, публикацию и публичный каталог.

## Публикация

В каталог попадают только организации, которые:

- отмечены как подрядчики;
- имеют статус `published`;
- доступны по уникальному `slug`.

Контакты на публичной странице показываются только авторизованным пользователям.

## Данные профиля

Организация хранит название, город, описание, сайт, логотип и статус. `contractor_profiles` хранит подробное описание, бюджет, цены, размер команды и контакты. Услуги связаны через `organization_services`.

## Маршруты

- `/contractors`
- `/contractors/[slug]`
- `/dashboard/contractor`
- `/dashboard/contractor/profile`

## Реализация

- `app/actions/contractor-profile.ts`
- `components/contractors/contractor-card.tsx`
- `components/contractors/contractor-filters.tsx`
- `lib/supabase/queries.ts`

## Важное

Публичный `slug` формируется из сайта или названия, но пользователь может отредактировать его. Уникальность должна проверяться до сохранения.
