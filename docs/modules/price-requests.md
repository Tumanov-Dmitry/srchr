# Запрос стоимости

## Назначение

Модуль собирает структурированный запрос клиента на оценку стоимости и позволяет управлять предложениями исполнителей.

## Основной сценарий

1. Пользователь заполняет запрос.
2. Запрос получает публичную или управляемую страницу.
3. Подходящие исполнители могут отвечать на запрос.
4. Автор управляет запросом и просматривает ответы.

## Доступ

Публичная часть ограничена опубликованными данными. Управление доступно создателю, участникам организации с подходящей ролью и администраторам.

## Маршруты

- `/price-requests`
- `/price-requests/new`
- `/price-requests/[id]`
- `/price-requests/[id]/manage`
- `/dashboard/price-requests`
