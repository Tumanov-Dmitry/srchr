# SRCHR Documentation

This directory contains product, architecture, and module documentation for SRCHR.

## Structure

- `architecture.md` - technical architecture and operational model.
- `product/` - product vision, roles, business model, pricing, reputation, recommendations, search, and analytics.
- `modules/` - current behavior of implemented product modules.

## Usage

For development tasks, read documentation in this order:

1. `AGENTS.md`
2. `docs/product/vision.md`
3. relevant `docs/product/*.md`
4. relevant `docs/modules/*.md`
5. existing implementation files

The codebase remains the source of truth when docs are outdated. Update docs when changing business behavior.
